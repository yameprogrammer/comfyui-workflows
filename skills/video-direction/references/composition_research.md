## 2. Emotion ← composition (synthesis table rationale)

Craft sources repeatedly pair:

- **Symmetry / level / balance** → calm, order, formality  
- **Asymmetry / diagonal / edge** → tension, power imbalance  
- **Negative space** → isolation, luxury, dread (void)  
- **Lead room** → future, motion, empathy with gaze  
- **Tight positive space** → intimacy or claustrophobia  
- **Frame-in-frame** → trapped, watched, portal  
- **Scale in frame** (with size code) → awe vs dominance  

These are **probabilistic craft defaults** for agents, not universal psychology laws.
