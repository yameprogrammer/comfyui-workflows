# Story craft gates (working set)

Condensed from widely cited craft (Pixar/Emma Coats working rules as used in top director agent skills) + emotional-plot practice.  
**Not** a substitute for a full screenwriting course — gates for short-form AI video.

## Story spine (one line each)

1. Once upon a time…  
2. Every day…  
3. One day…  
4. Because of that…  
5. Because of that…  
6. Until finally…  

## Emotional engine (interrogate every concept)

| Element | Question |
|---------|----------|
| WANT | External conscious goal? |
| NEED | What they must learn (may conflict with want)? |
| WOUND / LIE | False belief the story pressures? |
| STAKES | Cost of failure — escalating? |
| TURN | One real reversal / recognition? |
| PEAK-END | Peak emotion + ending engineered? |

## Causal glue

Beats join with **but / therefore**, never endless **and then**.

## Coincidence rule

Coincidences that get characters **into** trouble: OK.  
Coincidences that get them **out**: cheating.

## Discount the obvious

Throw away the first 3–5 cliché images for a lyric/keyword. The sixth is often the film.

## Why this story

If generic in → generic out. Name the **specific** belief or paradox.

## Short-form beat budget

~1 visual beat per 2–4 seconds.  
60s ≈ 12–18 beats max; cut mercilessly.
