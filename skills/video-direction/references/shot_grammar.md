## Size codes

| Code | Name | Use | Abuse |
|------|------|-----|-------|
| ELS | Extreme long | World, scale, loneliness | Lose subject |
| LS / Wide | Long | Space relations, city, flood | Emotion diluted |
| FS | Full body | Walk, wardrobe, gesture | |
| MLS | Medium-long | Performance medium | |
| MS | Medium (waist) | Dialogue default | **no 3× row** |
| MCU | Medium close | Emotional approach | |
| CU | Close-up | Eyes/mouth confession | chorus-only spam ban |
| ECU | Extreme close | Motif skin/detail | identity risk |
| Insert | Detail | Props, hands, signs | **not face substitute** |
| POV / OTS | Point of view / over shoulder | Subjective | anatomy risk |
| Two-shot | Two subjects | Relationship | |

## Angles

Eye (empathy) · Low (power/hero) · High (small/surveillance) · Dutch (unease, rare) · Top-down (ritual/feet/water)

## Moves (pick one)

Static · Slow push-in · Pull-out · Pan · Tilt · Track/dolly · Handheld micro · Orbit (rare)

## Coverage ladder (every scene ideally)

1. Master / establishing  
2. Medium coverage (body/action)  
3. Close coverage (emotion)  
4. Insert / cutaway (motif/time)  
5. Special hero angle  

## Rhythm hard rules

- **R1** Adjacent: change size **or** angle  
- **R2** No same size code 3 consecutive  
- **R3** 12+ shots: wide, medium, CU/ECU, insert each ≥1  
- **R4** Axis/eyeline in continuous space  
- **R5** Motif match-cuts  
- **R6** Hook/chorus = visual event  

## 13-thumbnail test

Imagine all shots as thumbnails. If two look interchangeable, redesign one.
